Как научиться верстать?

Чтобы научиться верстать, необходимо выучить HTML и Css и далее много-много практиковаться.

Обычно это занимает примерно 3 месяца активного обучения.

Что для этого нужно?

Название Описание
HTML Делает разметку сайта
Css Стилизует элементы из HTML
